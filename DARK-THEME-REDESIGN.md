- **Background**: Dark navy/deep blue (`hsl(224 71% 4%)`)

- **Border**: Subtle dark borders (`hsl(222 47% 14%)`)

## 🔧 Technical Implementation

### CSS Features Used
```css
- backdrop-blur-xl: Glassmorphism
- bg-gradient-to-r: Gradient backgrounds
- animate-pulse: Pulse animations
- animate-float: Custom floating animations
- bg-clip-text: Gradient text
- shadow-primary/25: Colored shadows
- border-primary/30: Transparent borders
- opacity-[0.02]: Fine-tuned opacity
```

### Tailwind Classes
```tsx
- bg-card/50: Semi-transparent cards
- hover:border-primary/50: Hover border glow
- group-hover:scale-110: Group hover effects
- transition-all duration-300: Smooth transitions
- backdrop-blur-sm: Glassmorphism
- bg-gradient-to-r from-primary to-blue-400: Gradients
```

### Custom Animations
```css
@keyframes float: Floating elements
@keyframes gradient: Animated gradients
.animate-float: 6s ease-in-out infinite
.animate-gradient: 3s ease infinite
```
